# Hercio Moreira Gusmao Portfolio

Personal developer portfolio for Hercio Moreira Gusmao, built as a fast single-page portfolio with multilingual content, GitHub repository auto-sync, editable resume data, live project links, and GitHub Pages deployment.

## Tech Stack

- React
- TypeScript
- Vite
- GitHub Pages
- GitHub Actions
- GitHub API repository sync
- JSON resume content files

## Live Site

https://herciomoreira3.github.io/

## Scripts

```bash
npm run dev
npm run sync:github
npm run typecheck
npm run build
```
